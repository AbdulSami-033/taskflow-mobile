# 📱 TaskFlow Journal – Android To-Do Application

TaskFlow Journal is a simple and intuitive Android application developed using Kotlin to help users organize their daily tasks and improve productivity. The app provides an easy way to create, manage, edit, and complete tasks through a clean and user-friendly interface.

---

## ✨ Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑 Delete tasks
- ✔ Mark tasks as completed
- 📋 View all saved tasks
- 📱 Clean and responsive Android UI
- 💾 Local data storage
- ⚡ Fast and lightweight performance

## 🚀 Getting Started

### Prerequisites

- Android Studio
- Android SDK
- JDK 17 or above

### Installation

Clone the repository

```bash
git clone https://github.com/AbdulSami-033/taskflow-mobile.git
```

Open the project in Android Studio and click **Run**.
